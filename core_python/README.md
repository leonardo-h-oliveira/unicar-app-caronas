# UniCar – Core Logic (Python)

Este diretório implementa a lógica central do UniCar em Python, baseada nas regras de negócio do protótipo.

Objetivo:
- representar o “coração” do sistema (vagas, entradas/saídas, cancelamento)
- demonstrar organização de código e clareza de regras
- permitir execução local via script de demonstração (sem interface)

Arquivos:
- `models.py` → estruturas de dados (Usuário/Carona)
- `services.py` → regras de negócio
- `repository.py` → persistência simples (JSON)
- `demo.py` → cenário de uso
