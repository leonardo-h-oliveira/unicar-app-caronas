# Origem Acadêmica do Projeto

O projeto **UniCar** foi desenvolvido como Trabalho de Conclusão de Curso (TCC)
do Bacharelado Interdisciplinar em Ciência e Tecnologia (BICT) da Universidade Federal de Alfenas (UNIFAL-MG),
campus Poços de Caldas.

**Autores:**
- Bruna Helena Antonialli Gomes
- Leonardo Henrique Oliveira

**Orientador:**
- Prof. Dr. Luiz Felipe Ramos Turci

O trabalho resultou em um protótipo funcional de aplicativo de caronas universitárias,
com desenvolvimento completo de fluxos, lógica de negócio, autenticação e integração
com banco de dados em nuvem (Firebase).

Ano: 2025
