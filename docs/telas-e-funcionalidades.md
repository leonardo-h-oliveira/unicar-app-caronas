# Telas e Funcionalidades do Sistema

## Screen 1 – Login e Registro
Responsável pelo cadastro e autenticação de usuários via Firebase Authentication.
Permite registro de usuário, cadastro de veículo e definição de pontos de parada.

Principais funções:
- Registro e login
- Armazenamento de dados do usuário
- Integração com Firebase
- Uso de variáveis globais para controle de sessão

## Screen 4 – Tela Principal
Tela de navegação central do aplicativo, permitindo:
- Oferecer carona
- Solicitar carona
- Encerrar aplicação

## Screen 2 – Solicitação de Carona
Permite ao usuário visualizar caronas disponíveis,
selecionar uma oferta e confirmar participação,
com atualização automática de vagas no Firebase.

## Screen 5 – Seleção de Ponto de Embarque
Tela responsável pela escolha do ponto de parada
e comunicação automática com o motorista via WhatsApp.

## Screen 3 – Oferta de Carona
Permite o cadastro de novas ofertas,
definindo horário, vagas e sentido da viagem,
com armazenamento estruturado no Firebase.
