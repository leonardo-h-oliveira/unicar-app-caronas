# Decisões Técnicas

## Plataforma: MIT App Inventor
Escolhida por acelerar o desenvolvimento do protótipo, permitindo foco em:
- fluxos do usuário
- regras de negócio
- validações
- integração com backend

## Backend: Firebase
Utilizado para:
- autenticação de usuários
- persistência e consulta de ofertas de carona
- atualização de disponibilidade de vagas

## Persistência local (quando aplicável)
Uso de armazenamento local para dados temporários de sessão e preferências do usuário.
